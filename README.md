# Portfolio — Divyansh Singh Patel

Personal portfolio website built with HTML, CSS, and JavaScript. Features a cyberpunk-inspired design with interactive animations, a journey timeline, certificates showcase, and project highlights.

## Sections

- **Hero** — Typing effect with role rotation, parallax gradient orbs, floating particles
- **About** — Bio and highlights with scroll-triggered reveals
- **Journey** — Interactive timeline with certificate lightbox links
- **Certificates** — Grid of achievement cards with detail lightbox
- **Projects** — Featured project showcase with tech tags and link grid
- **Skills** — Categorized skill set display
- **Contact** — Social links grid with hover effects (GitHub, Instagram, LinkedIn, X, Modrinth, CurseForge, Telegram, Email)

## Features

- Responsive design (mobile-friendly navigation)
- Lightbox for certificate details and images
- Scroll reveal animations with multiple variants
- Card tilt on hover
- Magnetic button effect
- Smooth scrolling navigation
- Active section tracking in navbar

## Tech Stack

- HTML5
- CSS3 (custom properties, glassmorphism, keyframe animations)
- JavaScript (IntersectionObserver, DOM manipulation)
